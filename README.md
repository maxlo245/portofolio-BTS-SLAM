# Portfolio Personnel

Ce projet est un site web de portfolio personnel, réalisé en HTML, CSS et JavaScript.

## Structure du projet
- `index.html` : page d'accueil
- `projets.html` : liste des projets
- `contact.html` : formulaire de contact
- `styles/` : fichiers CSS
- `scripts/` : fichiers JavaScript
- `images/` : images et assets

## Démarrage
Ouvrez le fichier `index.html` dans votre navigateur pour visualiser le site.

## Personnalisation
- Remplacez les textes et images par vos propres informations.
- Modifiez le style dans `styles/main.css` selon vos préférences.
